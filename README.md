# Todo Tracker (MERN)

> Full stack Todo tracker

## Usage

```
change config.env file in config folder
```

```
 yarn install
 cd client yarn install
 cd ..

 # Run front and backend
 yarn run dev

 # Backend only
 yarn run server

 # Frontend only
 yarn run client

 # Build client
 cd client
 yarn run build

 # Prod
 yarn start
```
